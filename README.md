# 漏水音判定アプリ

音声ファイルを読み込み、Gemini APIに送信して漏水音かどうかを判定する簡易Webアプリです。

## 使い方

1. Gemini APIキーを用意します。
2. 静的サーバーで `app/index.html` を開きます。
3. APIキーと音声ファイルを入力し、「解析する」を押します。

## ローカル起動例

```bash
cd app
python -m http.server 8000
```

ブラウザで `http://localhost:8000` を開きます。

## 注意

- 音声ファイルは最大20MBまでを想定しています。
- Geminiの返答は参考情報です。現場確認とあわせて判断してください。
